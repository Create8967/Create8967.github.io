What is Arduino

Arduino is a programmable computer. But it is just a board.

The arduino is usually used for elecrtical circuits so it would be good for circuit programming.

In this tutorial you will learn about how to make circuits and much more. But you will need some parts.

The parts

Before you do anything you're going to need some parts.

First you may want to get an Arduino kit.

It is much easyer that getting all of the parts seperate.

Here is a list of other parts you may need.

  • Some motors
  • A speaker

The software

You are going to need the Arduino software for programming.

You can go to the download page on the Arduino website by clicking that Arduino image 6 times.

Or if it doesn't work. You can click on the link here.

Remember to get the version for your computer. If you have Windows 32 or 64 bit get the download for that. Or if you have a Mac get the one for Mac.

Once you have done all of that. It is time to start making stuff.
